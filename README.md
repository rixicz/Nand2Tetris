# Nand2Tetris Project
## The goal of this project is to build a functional computer that can execute a moderately complex program (e.g. Tetris).
This project is an example that was build along a course on Coursera from the Hebrew University of Jeruzalem. 
It is divided into two major parts containing 12 projects in total.
